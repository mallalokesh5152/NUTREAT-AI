# 🌱 NUTREAT AI

## AI-Based Seasonal Food Recommendation System

A smart prototype website that suggests suitable foods, traditional drinks, and nourishment practices based on weather, season, region, age group, lifestyle, temperature, and humidity.

### 🎯 Features

- **Weather-Based Recommendations** - Uses temperature and humidity to understand climate conditions
- **Traditional Knowledge** - Includes Indian seasonal foods, drinks, and nourishment practices
- **Interactive Chatbot** - Real-time food suggestions based on user inputs
- **Multi-Parameter Analysis** - Considers season, region, age, lifestyle, temperature, and humidity
- **Responsive Design** - Works seamlessly on desktop and mobile devices

### 🔧 System Workflow

```
User Inputs → Weather Check → Recommendation Logic → Food Suggestions
```

### 📋 Input Parameters

- **Location/Region** - Geographic area for localized recommendations
- **Season** - Summer, Monsoon, or Winter
- **Age/Age Group** - For age-specific nutritional needs
- **Lifestyle** - Low Activity, Moderate Activity, or High Activity
- **Temperature** - Current temperature in °C
- **Humidity** - Current humidity percentage

### 🍽️ Recommendation Categories

1. **Suggested Foods** - Season and weather-appropriate food items
2. **Traditional Drinks** - Seasonal beverages for optimal health
3. **Nourishment Practices** - Tips and dietary recommendations
